# Salon Sage

A modern mobile application that streamlines salon appointment booking, replacing manual methods with a digital, efficient, and user-friendly system.

## 📱 App Preview

<img src="main screen.png" width="300" alt="Salon Sage Main Screen">

## ✨ Features

- User Authentication
- Salon Listings & Discovery
- Service Selection
- Appointment Booking & Management
- Ratings & Reviews
- Push Notifications

## 🛠️ Tech Stack

- **Framework:** Flutter
- **Design:** Figma (Mobile-first approach)
- **State Management:** Provider / GetX

## 🚀 Future Enhancements

- Secure online payment integration
- Real-time availability tracking
- GPS-based salon discovery
- Advanced search filters

## 📋 Prerequisites

- Flutter SDK
- Android Studio / VS Code
- iOS Simulator or Android Emulator

## 🔧 Installation

```bash
# Clone the repository
git clone https://github.com/ShaistaS1/salon-sage.git

# Navigate to project directory
cd salon-sage

# Get dependencies
flutter pub get

# Run the app
flutter run
